🚗 Tire Wear Prediction System

This project is an end-to-end Machine Learning application that predicts tire wear percentage based on vehicle and driving-related features. The system includes data preprocessing, feature engineering, model training, and a frontend interface for real-time predictions.

🔹 Key Features

Data preprocessing and cleaning pipeline

Feature ranking and selection

Model training and evaluation

Scaler and preprocessing object serialization

Frontend interface for prediction

Modular backend architecture

🔹 Tech Stack

Python

Pandas, NumPy

Scikit-learn

Streamlit / Flask

Joblib / Pickle

🔹 Project Architecture

backend/ – Data generation, preprocessing, feature ranking, model training

frontend/ – User interface for prediction

data/ – Raw and processed datasets

assets/ – Static resources

🔹 Workflow

Raw data ingestion

Data preprocessing and feature engineering

Model training and evaluation

Model serialization

Real-time prediction via frontend
